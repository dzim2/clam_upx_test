Recommended test versions

VersionAnomaly
1.20No method flags; NRV2E absent; win64/pe and linux/amd64 absent. Will silently fall back to NRV2B or NRV2D based on size heuristic.
2.02LZMA absent. --lzma flag will error. linux/amd64 present but NRV2E stub may be limited to certain arch combinations — confirm against specific target before relying on it.
3.09LZMA stubs present for all arches but no --lzma CLI flag. LZMA can only be invoked via --brute/--ultra-brute or by patching opt->method directly. win64/pe absent.
4.20 / 5.01 / 5.11Full capability. Differences between these three are minor (riscv64, staged backends, bug fixes). For a stable test corpus, 4.20 is the oldest version covering all major use cases.
